데이터베이스 파일
DBMS는 데이터베이스에 저장된 데이터와 이를 관리하면서 발생하는 부수적인 데이터를 파일로 나눠 저장하고 이를 한데 묶어 하나의 데이터베이스를 관리한다.

1. 데이터 파일:실제 데이터를 저장한다.
2. 로그 파일: 데이터베이스에서 수행한 모든 변경사항[ex.데이터 생성,수정,삭제 작업 등]을 저장한다.
3. 인덱스 파일:데이터베이스에 내 검색 속도를 향상하기 위해 필요한 정보를 저장한다.
4. 설정 파일:DBMS의 구성과 운영 설정을 저장한다.
5. 메타데이터 파일:데이터베이스 구조[예.테이블 구조, 데이터 유형, 관계 등]에 대한 정보를 저장한다.

테이블
데이터베이스의 데이터 파일은 '테이블에 담아 데이터를 저장한다.'
- 일련의 데이터를 열과 행으로 저장하는 틀인데 각 열은 해당 데이터의 속성을 나타내고, 각 행은 속성이 모여 만든 하나의 데이터 항목을 나타난다.
